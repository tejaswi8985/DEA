# 🚦 Traffic Congestion Analysis Dashboard

A data-driven smart city project that analyzes and visualizes traffic congestion patterns using Exploratory Data Analysis (EDA), Machine Learning, and an interactive Streamlit dashboard.

---

## 📌 Project Overview

Urban traffic congestion is a major challenge in modern smart cities. This project focuses on analyzing traffic patterns using data exploration techniques and applying machine learning models to predict congestion levels.

The system provides insights into:

* Peak traffic hours
* Congestion hotspots
* Weekday vs weekend trends
* Traffic anomalies
* Future congestion predictions

---

## 🎯 Objectives

* Perform data cleaning and preprocessing of traffic data
* Analyze temporal (time-based) traffic patterns
* Identify congestion-prone zones
* Compare different machine learning models
* Build an interactive dashboard for visualization

---

## 🧠 Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* Statsmodels (ARIMA)
* Plotly
* Streamlit

---

## 📊 Features

### 🔍 Data Analysis

* Hourly traffic trend analysis
* Zone-wise congestion analysis
* Heatmaps for traffic intensity

### 🤖 Machine Learning

* Linear Regression
* Random Forest
* XGBoost
* Model comparison using MSE

### 📍 Clustering

* K-Means clustering to identify traffic patterns

### 📈 Forecasting

* ARIMA model for congestion prediction

### 🎨 Dashboard

* Dark theme UI
* Interactive charts (Plotly)
* Sidebar filters
* Real-time predictions
